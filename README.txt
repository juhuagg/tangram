大作业压缩包中包含五个文件：
1、算法部分【TangramDFS;TangramBFS】
（1）若需要批阅程序的算法建议，建议运行tangtamDFS文件夹中的tangramfinal项目，
tangramBFS文件除搜索策略，算法和DFS完全相同，但速度较慢；
（2）项目编译环境 ：VS2017x64 debug，需要预先配置OpenCV；
（3）tangramDFS文件内含有picdata图片库，内含18张图片，编译运行前需要在主函数改
变图片读取路径，选取一张图片读取；
（4）调试运行后，每按一次回车，程序会显示当前搜索节点图像，直至搜索到解答后显示
解答图像；
（5）7、8、13、14号图像搜索路径较短，5号图像搜索路径较长；

2、QT部分【QTUI】
（1）QT部分代码被打包在QTUI文件夹中，使用QT5运行test.pro即可查看项目；
（2）QT部分核心算法代码直接从VS黏贴，较为凌乱，不建议在QT中查看核心算法；

3、可执行文件【Tangramdemo】
（1）可执行文件在tangramdemo文件中，其中tangramsolve/tangramsolve_boxed都可
直接运行
（2）图片库picdata同样被打包在tangtamdemo文件夹子中供用户选择
（3）可执行文件具体操作流程见报告
